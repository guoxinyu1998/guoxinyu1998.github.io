Install instructions

To install this package:

Extract this package in <web site>/img/coin/. If your site is http://www.example.com, you should be able to access a file named http://www.example.com/img/coin/favicon.ico.

Insert the following code in the head section of your pages:

<link rel="apple-touch-icon" sizes="180x180" href="/img/coin/apple-touch-icon.png">
<link rel="icon" type="image/png" sizes="32x32" href="/img/coin/favicon-32x32.png">
<link rel="icon" type="image/png" sizes="16x16" href="/img/coin/favicon-16x16.png">
<link rel="manifest" href="/img/coin/site.webmanifest">
<link rel="mask-icon" href="/img/coin/safari-pinned-tab.svg" color="#5bbad5">
<link rel="shortcut icon" href="/img/coin/favicon.ico">
<meta name="apple-mobile-web-app-title" content="Guo Xinyu's Blog">
<meta name="application-name" content="Guo Xinyu's Blog">
<meta name="msapplication-TileColor" content="#2b2b2b">
<meta name="msapplication-config" content="/img/coin/browserconfig.xml">
<meta name="theme-color" content="#ffffff">
