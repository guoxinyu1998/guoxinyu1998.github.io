Siamese Neural Networks for One-Shot Image Recognition

Abstract

为机器学习应用程序学习好的特征的过程在计算上可能非常昂贵，并且在数据很少的情况下是困难的。这方面的一个典型例子是one-shot learning环境，在这个环境中，我们必须对每一个新类的一个例子做出正确的预测。在本文中，我们探索了一种学习Siamese神经网络的方法，该网络采用独特的结构来自然排列输入之间的相似性。一旦对网络进行了调整，我们就可以利用强大的鉴别特征，将网络的预测能力推广到新数据和来自未知分布的全新类别。使用卷积结构，我们能够获得比其他深度学习模型更强的结果，在one-shot分类任务上具有接近最先进的性能。

1.Introduction

1.1Overview

人类表现出很强的获取和识别新模式的能力。特别是，我们观察到，当刺激物出现时，人们似乎能够快速理解新概念，然后在未来的知觉中识别出这些概念的变化。这个原则适用于更广泛的领域，而不仅仅是泛化特定概念类的不可见图像。如果给一个人看一个特定物种的图像，比如一只老虎，他不仅能够用这张图像来识别新的老虎图像，而且还能对新物种的图像做出有根据的猜测。例如，我们可以断言，与其他哺乳动物相比，狮子与老虎更相似。因此，这种类型的推理可能非常有用，但它不一定是我们的模型中的固有属性，它可能非常适合可用的类，而不需要学习其他类的结构。

当被迫对很少有监督信息可用的数据进行预测时，机器学习中的算法经常会崩溃。我们希望在不需要进行大量再培训的情况下归纳出这些不熟悉的类别。由于数据有限或在在线预测设置(如web检索)中，再培训可能是昂贵的或不可能的。

one-shot学习的问题可以通过开发领域特定的特征或对目标任务具有高度区分性的推理过程来直接解决。因此，包含这些方法的系统往往在类似的实例中表现出色，但不能提供可普遍应用于其他类型问题的健壮解决方案。在本文中，我们提出了一种新的方法，它限制了对输入结构的假设，同时自动获取特征，使模型能够从少数例子中成功地泛化。我们建立在深度学习框架的基础上，该框架使用多层非线性来捕获输入空间中转换的不变性，通常通过利用一个具有许多参数的模型，然后使用大量数据来防止过拟合。这些特征非常强大，因为我们能够在不强加强先验的情况下学习它们，尽管学习算法本身的代价可能是相当大的。

1.3方法

将主要目标放在字符识别上，尽管通用方法几乎可以用于任何形式(下图所示)。对于这个领域，我们使用大型Siamese卷积神经网络，它:A)能够学习通用图像特征，用于预测未知的类分布，即使这些新分布的样例很少;B)很容易使用标准优化技术对从源数据中采样的数据对进行训练;C)提供一种具有竞争力的方法，不依赖于特定领域的知识，而是利用深度学习技术。

![1](D:\研究生\文献\FSL in NLP\siamese neural nettworks for one shot image recogntion\1.png)

我们的总体战略。1)训练模型来区分相同/不同pairs的集合。2)根据所学的特征映射进行归纳，以评估新类别，从而进行验证

为了建立一种one-shot图像分类模型，我们的目标是首先学习一种能够区分图像pairs类别身份的神经网络，这是图像识别的标准验证任务。也就是说，给定任意两个来自相同字母的图像(或者可能是不同的字母，尽管我们不考虑这种情况)，这个网络将预测这些图像是否描绘了相同的字符。

我们假设在验证方面表现出色的网络应该推广到One-shot分类。验证模型根据输入pairs属于同一类或不同类的概率来识别输入对。然后，该模型可以用于以成对的方式对照单独的测试图像来评估新图像。然后，根据验证网络，得分最高的配对被授予分类任务的最高概率。如果通过验证模型学习到的特征足以确认或否认从一组字母中提取的字符的身份，那么它们应该足以用于其他字母

通过在神经网络的最后一层选择一个好的度量，然后对参数化施加一定的约束，我们应该能够学习这种表示。注意，我们不建议学习度量，所以我们的方法只专注于特征学习。一般来说，这个想法是首先通过一个基于度量的监督方法学习好的表示，然后重用网络的特征进行one-shot learning，而不需要任何再训练。

2.The Omniglot Dataset

2.1Overview

Omniglot包含了50种字母，从成熟的国际语言(如拉丁语和韩语)到鲜为人知的当地方言。它还包括一些虚构的字符集。

每个字母表中的字母数量相差很大，从大约15个到40个以上的字符.

All characters across these alphabets are produced a single time by each of 20 drawers.

莱克将数据分为40个字母background set和10个字母evaluation set。我们保留这两个术语是为了区别于正常的训练、验证和测试集，这些测试集可以从背景集中生成，以便调整模型进行验证。背景集用于通过学习超参数和特征映射来开发模型。相反，评估集仅用于衡量一次性分类性能。在本文中，我们使用的术语“背景集”和“评估集”与莱克的工作中引用的术语完全一致。

2.2One-shot learning task

为了凭经验评估one-shot learning效果，莱克开发了一个20-ways字母的分类任务，首先从为评估集保留的字母中选择一个alphabet，连同随机选择的20个characters。20个drawers中的两个也是从评估drawers池中选择的。然后，这两个drawer产生了20个characters的样本。由第一个drawer产生的每个characters被表示为测试图像，并与来自第二个drawer的所有20个字符进行单独比较，目的是从第二个drawer的所有字符中预测对应于测试图像的类别。

这一过程对所有alphabets重复两次，因此十个评估alphabets中的每一个都有40次one-shot learning试验。这构成了总共400次one-shot learning试验，从中计算标准分类精度。

3.Image Verification with Siamese Neural Nets

3.1 Overview

Siamese神经网络由两个网络组成，这两个网络接受不同的输入，但在顶部由一个energy function连接。该函数计算每侧最高级别特征表示之间的某种度量(下图所示)。Siamese网络之间权重共享。

![2](D:\研究生\文献\FSL in NLP\siamese neural nettworks for one shot image recogntion\2.png)

图（全连接层）：一个简单的2隐藏层siamese网络，用于二进制分类和逻辑预测p。该网络上下两部分相同，形成双网络，在每一层都有共享的权重矩阵。

h是特征向量 d是距离向量 $d_{l}=|h_{1,l}-h_{2,l}|$

对d进行加权求和，经过sigmoid后，转化为概率p

这个策略有两个关键点：

1.它确保了预测的一致性。权重捆绑保证了两幅极其相似的图像不可能被它们各自的网络映射到特征空间中非常不同的位置，因为每个网络计算相同的函数。

2.网络是对称的

在本文中，我们使用twin特征向量 h1 和 h2 之间的加权L1距离结合sigmoid激活，其映射到区间[0，1]。因此，交叉熵loss是训练网络的自然选择。

3.2 Model Definition

我们的标准模型是一个siamese神经网络，它有L个全连接层，每个层都有$N_{l}$个神经单元，其中$h_{1,l}$表示first twin 中$l$层的隐藏向量，$h_{2,l}$表示second twin中$l$层的隐藏向量。在前$L-1$层中，我们使用(ReLU)单元，因此对于任何一层$l \in {1,…,L-1}$:
$$
h_{1,m} = max(0,W_{l-1,l}^{T}h_{1,(l-1)} + b_{l}) \
h_{2,m} = max(0,W_{l-1,l}^{T}h_{2,(l-1)} + b_{l})
$$
其中$b_{l}$是共享偏置向量，$W_{l-1,l}$是连接$l-1$层$N_{l-1}$个神经单元和$l$层$N_{l}$个神经单元的，大小为$N_{l-1} \times N_{l}$的权重矩阵:
$$
W_{l-1,l} = \left[
\matrix{
w_{1,1} ,w_{1,2},…w_{1,N_{l}}\
w_{2,1},w_{2,2},…w_{2,N_{l}} \
…,…,…,… \
w_{N_{l-1},1},…,…,w_{N_{l-1},N_{l}}
}
\right]
$$

在第(L-1)个feed-forward层之后，我们通过固定距离函数$p = \sigma (\sum_{j}\alpha_{j}|h_{1,l}^{j} - h_{2,l}^{j}|)$比较每个Siamese计算的特征，其中$\sigma$是sigmoid激活函数。最后一层在第(L-1)个隐藏层的学习特征空间上引入度量，并对两个特征向量之间的相似性进行评分。$\alpha_{j}$是模型在训练过程中学习的附加参数，在距离上的权重。这为连接两个siamese网络定义了最终的第L个全连接层。

3.3Learning

损失函数

M表示minibatch的大小，用$i$来索引第$i$次minibatch。现在让$y(x_{1}^{(i)},x_{2}^{(i)})$是一个长度为M的向量，它包含了minibatch的标签，这里我们假设如果$x_{1}$和$x_{2}$来自同一个character,则$y(x_{1}^{(i)},x_{2}^{(i)})=1$，否则$y(x_{1}^{(i)},x_{2}^{(i)})=0$。我们将一个正则化的交叉熵目标加到我们的二元分类器，其形式如下:
$$
L(x_{1}^{(i)},x_{2}^{(i)}) = y(x_{1}^{(i)},x_{2}^{(i)})logp(x_{1}^{(i)},x_{2}^{(i)}) + (1-y(x_{1}^{(i)},x_{2}^{(i)}))log(1-p(x_{1}^{(i)},x_{2}^{(i)}))+\lambda^{T}|w|^{2}
$$
优化：

该目标与标准反向传播算法相结合，在标准反向传播算法中，由于捆绑的权重，双网络之间的梯度是可加的。固定minibatch的大小为128,学习率$\eta_{j}$，动量$\mu_{j}$,逐层定义的$L_{2}$正则化权重$\lambda_{j}$，因此我们在epoch T的更新规则如下:
$$
w_{kj}^{T}(x_{1}^{(i)},x_{2}^{(i)}) = w_{kj}^{T} + \Delta w_{kj}^{T}(x_{1}^{(i)},x_{2}^{(i)}) + 2\lambda_{j}|w_{kj}| \
\Delta w_{kj}^{T}(x_{1}^{(i)},x_{2}^{(i)}) = -\eta_{j}\nabla w_{kj}^{T} + \mu_{j}\Delta w_{kj}^{(T-1)}
$$
其中$\nabla w_{kj}$是某层的第j个神经元与连续层的第k个神经元之间权值的偏导数。

权重初始化

我们从一个标准差为$\frac {1}{fan-in}$的零均值正态分布中得到网络权值,将$fan-in$定义为

网络中特定层中每个神经元的输入权值的个数。偏差是平均值为0.5，固定方差为$10^{-2}$的高斯分布。

学习过程

虽然我们允许每一层有不同的学习速率，但是学习速率在整个网络中每一epoch均匀衰减1%，因此$\eta_{j}^{T+1} = 0.99\eta_{j}^{T-1}$。我们发现，通过对学习速率进行退火（anneal），网络能够更容易地收敛到局部最小值。我们将每层的动量初始值设为0.5，每一epoch线性增加，直到达到值$\mu_{j}$，即第j层的单个动量项。

我们对每个网络进行了最多300次epoch，但在一组320个one-shot学习任务中监控了one-shot验证错误，这些任务是从验证集中的字母和drawers中随机生成的。当验证误差在20个epoch内没有减小时，我们根据one-shot验证误差停止使用模型在最佳历元的参数。如果验证错误在整个学习计划中持续减少，我们会保存此过程生成的模型的最终状态。

用小的仿射失真增强了训练集

3.4实验结果

实验中，使用了一个三层架构，其中有两个完全连接的层，然后是一个连接siamese network的L1距离度量。Then we evaluated a linear combination of the component-wise distances and mapped the activation to a category with a single sigmoid unit.

用30，000、90，000和150，000个训练例子组合了三个不同的数据集大小。我们留出总数据的60%用于训练。Omniglot数据集不是为验证而设置的，因此我们生成了自己的标记示例。最基本的过程包括为一个字母表挑选两个字符类和两个drawers。然后，我们将这些图像组合起来，形成一个匹配对和一个不匹配对。

我们发现，3层网络提供了可靠的性能而没有过度拟合；这只有在大数据集、heavy L2正则化和数据扩充的情况下才能实现。这种大于3层的全连接网络提供了递减的收益，并且没有学到明显更好的特性。然而，我们的工作在这一类别中相当有限，我们没有使用我们的自动超参数选择来选择层数或神经元配置(例如激活功能)，这对于未来的运行是一个可能的改进。

4.Learning Deep Convolutional Feature Hierarchies

4.1Overview

已经有了一个标准的feedforward架构，在Siamese网络的每一侧使用捆绑的权重，但是所有的神经元都不加区分地连接在一起，没有任何本地连接。在这里，我们描述了前一节中提出的模型的直接扩展，以包括卷积层。

![4](D:\研究生\文献\FSL in NLP\siamese neural nettworks for one shot image recogntion\4.png)

图：为验证任务选择的最佳卷积结构。siamese twin没有被描绘，但是在有4096个单元的全连接层之后立即连接，在那里计算矢量之间的L1距离。

4.2模型定义

使用一种相当传统的卷积结构。该模型由一系列卷积层组成，每个卷积层使用一个具有不同大小、固定步长1的单通道的滤波器。卷积滤波器的数量被指定为16的倍数，以在GPU内核中提供更好的性能。网络在输出特征映射上应用ReLU激活函数。随后是窗口为$2 \times 2$的max-pooling。因此，每层中的第k个过滤器映射采用以下形式:
$$
h_{1,m}^{(k)} = max-pool(max(0,W_{l-1,l}^{(k)} \star h_{1,(l-1) } + b_{l}),2) \
h_{2,m}^{(k)} = max-pool(max(0,W_{l-1,l}^{(k)} \star h_{2,(l-1) } + b_{l}),2)
$$
$\star$是卷积操作

$W_{l-1,l}$是表示l层特征映射的三维张量(可以看成是滤波器)

最终卷积层中的单元被展平为单个向量。这个卷积层之后是一个全连接层，然后是一个计算每个siamese twin之间距离度量的层，这个度量被提供给一个单个的sigmoid输出单元。

4.3Learning

使用固定权重的初始化方案，而不考虑网络每层的神经元数量，不使用fan-in(fan-in指的是某层中每个神经元的输入权重的个数)

从均值为0,标准差为$10^{-2}$的正态分布初始化卷积层中的所有网络权重

偏差也从正态分布开始，均值为0.5，标准差为$10^{-2}$。

在全连接层中，偏差以与卷积层相同的方式初始化，但是权重从零均值和标准差为$2 \times 10^{-1}$的更宽的正态分布中提取

对于较大的网络，我们将训练时间限制在200个epoch，而不是300个epoch，因为卷积网络的训练时间可能要长得多。

4.4实验结果

使用Whetlab为模型的一些关键方面选择超参数，如学习速率、动量以及卷积滤波器的数量和大小

从验证任务中两个表现最好的网络中提取了前32个过滤器。虽然过滤器之间有一些共同作用，但很容易看到一些过滤器相对于原始输入空间承担了不同的角色。（过滤器适应不同的角色:一些寻找非常小的点状特征，而另一些功能像大规模的边缘检测器。）
