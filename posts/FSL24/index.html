ABSTRACT

  • 提出利用基于推理的图模型来研究小样本学习问题，输入图像标签或者隐藏标签。通过使用消息传播算法，定义了一种图神经网络结构来解决FSL。
  • 除了提高数值性能之外，框架还可以轻松扩展到小样本学习的变体，例如半监督学习或主动学习，这证明了基于图的模型能够很好地处理“关系”任务。

INTRODUCTION

Few Shot learning部分方法，MN、PN、RN……

MN方法将 few shot learning 任务转换为监督分类问题，将一组support set 图像映射到所需标签中，并开发了一种end-to-end的方法，通过注意力机制接收support sets作为输入……，该论文的方法在MN的基础上，并认为Few-Shot中的任务可以自然的表示为一个图上的监督插值问题，其中nodesnodesnodes和数据集中的图像相对应，边edgesedgesedges 由一个可以训练的kernels给出(本文用的是MLPMLPMLP)。

kernels

利用图结构化数据表示学习的最新进展。提出了一个基于图的学习模型，该模型实现了任务驱动的消息传递算法。 最终的架构经过end-to-end的训练，捕获了任务的不变性，并在简单性，通用性，性能和样本复杂性之间做了很好的权衡。

贡献：

  • 将few shot learning 作为有监督的消息传递任务，使用图神经网络进行end-to-end训练。
  • 在Omniglot和Mini-Imagenet上用最少的参数实现了最好的性能
  • 扩展到半监督学习和主动学习

FEW-SHOT

在两个数据集上执行不同的q−shot K−wayq-shot \ K-wayq−shot K−way实验，从数据集中随机抽取KKK个类，从每个类中随机抽取qqq个样本。从KKK个样本中随机选取一个额外样本进行分类。

注：和episodeepisodeepisode 不一样，该实验的数据处理是这样的：

  • 随机抽取KKK个类别，每个类别指定一个序号，作为本次task数据的label，在其中随机指定一个类别作为positive_class
  • 从每个类别中随机抽取qqq个数据，其中positive_class类别需要多抽取一个数据，即q+1q + 1q+1个
  • 假设batch_size为16，则每个batch中有16个图，每个图有K×q+1K \times q + 1K×q+1个nodes
  • K×qK \times qK×q个样本是有label的，可以得到对应的one-hot编码，再连接特征向量，输入到图中。对于多出来的一个样本，假设标签是未知的，用均匀分布1/K1/K1/K (K−wayK-wayK−way)取代one-hot表示，连接特征表示，将所有数据输入到图中。
  • 目的是建立K×q+1K \times q +1K×q+1个数据之间的联系，最终对标签未知的一个样本进行预测
