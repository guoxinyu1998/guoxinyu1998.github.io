Abstract

  1. 在数据很少的情况下，很难为machine learning程序 学习到好的特征。
  2. 一个典型例子是one-shot learning，在这个环境中，我们必须对每一个新类的唯一样本做出正确的预测。
  3. 本文探索了一种学习Siamese神经网络的方法，该网络采用独特的结构来比较输入之间的相似性。

Introduction

  1. 人类具备很强的获取和识别新模式的能力

    当刺激物出现时，人们似乎能够快速理解新概念，然后在未来的知觉中识别出这些概念的变化。如果给一个人看一个特定物种的图像，比如一只老虎，他不仅能够用这张图像来识别新的老虎图像，而且还能对新物种的图像做出有根据的猜测。例如，我们可以断言，与其他哺乳动物相比，狮子与老虎更相似。

  2. one-shot learning

    在有监督的学习中，若可用的数据量过少，机器学习中的算法经常会崩溃。one-shot学习的问题可以通过开发领域特定的特征或对目标任务具有高度区分性的推理过程来直接解决。因此，包含这些方法的系统往往在类似的中表现出色，但不能提供可普遍应用于其他类型问题的健壮解决方案。在本文中，我们提出了一种新的方法，它限制了对输入结构的假设，同时自动获取特征，使模型能够从少数样本中成功地泛化。我们建立在深度学习框架的基础上，该框架使用多层非线性来捕获输入空间中转换的不变性，通常通过利用一个具有许多参数的模型，然后使用大量数据来防止过拟合。这些特征非常强大，因为我们能够在不强加强先验的情况下学习它们，尽管学习算法本身的代价可能是相当大的。

略

任务目标：字符识别

孪生卷积网络优势：

  • 能够学习通用图像特征，用于预测未知的类分布，即使这些新分布的样例很少;
  • 很容易使用标准优化技术对从源数据中采样的数据对进行训练;
  • 提供一种具有竞争力的方法，不依赖于特定领域的知识，而是利用深度学习技术。

总体战略：

  • 训练模型来区分相同/不同pairs的集合。
  • 根据所学的特征映射进行归纳，以评估新类别，从而进行验证
详细说明

为了建立一种one-shot图像分类模型，我们的目标是首先学习一种能够区分图像pairs类别身份的神经网络，这是图像识别的标准验证任务。也就是说，给定任意两个来自相同字母的图像，这个网络将预测这些图像是否描绘了相同的字符。

假设在验证方面表现出色的网络应该推广到One-shot分类。验证模型根据输入pairs属于同一类或不同类的概率来识别输入对。然后，该模型可以用于以成对的方式对照单独的测试图像来评估新图像。然后，根据验证网络，得分最高的配对被授予分类任务的最高概率。如果通过验证模型学习到的特征足以确认或否认从一组字母中提取的字符的身份，那么它们应该足以用于其他字母

通过在神经网络的最后一层选择一个好的度量，然后对参数化施加一定的约束，我们应该能够学习这种表示。注意，我们不建议学习度量，所以我们的方法只专注于特征学习。一般来说，这个想法是首先通过一个基于度量的监督方法学习好的表示，然后重用网络的特征进行one-shot learning，而不需要任何再训练。

Image Verification with Siamese Neural Nets

Siamese神经网络由两个网络组成，这两个网络接受不同的输入，但在顶部由一个energy function[1]连接。该函数计算每侧最高级别特征表示之间的某种度量(下图所示)。Siamese网络之间权重共享。

图（全连接层）：一个简单的2隐藏层siamese网络，用于二进制分类和逻辑预测p。该网络上下两部分相同，形成双网络，在每一层都有共享的权重矩阵。

h是特征向量 d是距离向量 dl=∣h1,l−h2,l∣d_{l}=|h_{1,l}-h_{2,l}|dl​=∣h1,l​−h2,l​∣

对d进行加权求和，经过sigmoid后，转化为概率p

这个策略有两个关键点：

  1. 它确保了预测的一致性。权重捆绑保证了两幅极其相似的图像不可能被它们各自的网络映射到特征空间中非常不同的位置，因为每个网络计算相同的函数。

  2. 网络是对称的

本文使用特征向量 h1 和 h2 之间的加权L1距离，结合sigmoid[2]将其映射到区间[0，1]。因此，交叉熵loss是训练网络最合适的选择。

Learning Deep Convolutional Feature Hierarchies

已经有了一个标准的feed-forward架构，在Siamese网络的每一侧使用捆绑的权重，但是所有的神经元都不加区分地连接在一起，没有任何本地连接。在这一节扩展了前一节中提出的模型(卷积层 + 全连接)。

图：为验证任务选择的最佳卷积结构。siamese twin没有被描绘，但是在有4096个单元的全连接层之后立即连接，在那里计算矢量之间的L1距离。

使用的是传统的卷积结构。该模型由一系列卷积层组成，每个卷积层使用一个具有不同大小、固定步长1的单通道的滤波器。卷积滤波器的数量被指定为16的倍数，以在GPU内核中提供更好的性能。网络在输出特征映射上应用ReLU激活函数。随后是窗口为2×22 \times 22×2的max-pooling。因此，每层中的第k个过滤器映射采用以下形式:

h1,m(k)=max−pool(max(0,Wl−1,l(k)⋆h1,(l−1)+bl),2)h2,m(k)=max−pool(max(0,Wl−1,l(k)⋆h2,(l−1)+bl),2)h_{1,m}^{(k)} = max-pool(max(0,W_{l-1,l}^{(k)} \star h_{1,(l-1) } + b_{l}),2) \\ h_{2,m}^{(k)} = max-pool(max(0,W_{l-1,l}^{(k)} \star h_{2,(l-1) } + b_{l}),2) h1,m(k)​=max−pool(max(0,Wl−1,l(k)​⋆h1,(l−1)​+bl​),2)h2,m(k)​=max−pool(max(0,Wl−1,l(k)​⋆h2,(l−1)​+bl​),2)

⋆\star⋆是卷积操作

Wl−1,lW_{l-1,l}Wl−1,l​是表示l层特征映射的三维张量(可以看成是滤波器)

最终卷积层中的单元被展平为单个向量。这个卷积层之后是一个全连接层，然后是一个计算每个siamese twin之间距离度量的层，这个度量被提供给一个单个的sigmoid输出单元。

使用固定权重的初始化方案，而不考虑网络每层的神经元数量，不使用fan-in(fan-in指的是某层中每个神经元的输入权重的个数)

从均值为0,标准差为10−210^{-2}10−2的正态分布初始化卷积层中的所有网络权重

偏差也从正态分布开始，均值为0.5，标准差为10−210^{-2}10−2。

在全连接层中，偏差以与卷积层相同的方式初始化，但是权重从零均值和标准差为2×10−12 \times 10^{-1}2×10−1的更宽的正态分布中提取

对于较大的网络，我们将训练时间限制在200个epoch，而不是300个epoch，因为卷积网络的训练时间可能要长得多。
